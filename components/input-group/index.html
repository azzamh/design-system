Input Group â

Combine multiple input in singleline.

Usage â

Simple Usage â

preview
vue
<template>
  <p-input-group>
    <p-input />
    <p-button variant="input">
      Submit
    </p-button>
  </p-input-group>
</template>

With Addon â

preview
vue
<template>
  <p-input-group>
    <p-input-group-addon>
      <IconEmail />
    </p-input-group-addon>
    <p-input />
  </p-input-group>
</template>

With Addon and Divider â

preview
vue
<template>
  <p-input-group>
    <p-input-group-addon>
      <IconEmail />

    <p-divider vertical />

    <p-button variant="input">
      Submit
    </p-button>
  </p-input-group>
</template>

Sizing â

Input Group also has 4 variants size: xs, sm, md, lg, default is md.

preview
vue
<template>
  <p-input-group size="xs">
    <p-input />
    <p-button variant="input">
      Submit
    </p-button>
  </p-input-group>
  <p-input-group size="sm">
    <p-input />
    <p-button variant="input">
      Submit
    </p-button>
  </p-input-group>
  <p-input-group size="md">
    <p-input />
    <p-button variant="input">
      Submit
    </p-button>
  </p-input-group>
  <p-input-group size="lg">
    <p-input />
    <p-button variant="input">
      Submit
    </p-button>
  </p-input-group>
</template>
